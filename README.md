### Run Locally
1. 打開終端機執行以下程序：
```
git clone https://github.com/Tim0124/react_native_job.git
```
2. 切換至當前資料夾，安裝套件
```
npm install
```
3. 啟動專案
```
expo-cli start --tunnel
```
4. 用手機掃描QR Code

5. 退出專案
```
control + c
```
